## Featured Projects

### 1. Meridian BI Agent
**AI-Powered Slack Bot for Real-Time Stock Data**

![Meridian BI Agent](screenshots/meridian-bi-agent.png)

An intelligent Slack bot that allows team members to query real-time stock prices using natural language. Instead of switching between apps or visiting financial websites, users simply ask questions like "@MeridianBI What's Apple's stock price?" and receive instant, accurate market data.

**Stack:**
- n8n (self-hosted on Railway)
- Groq LLM (for natural language understanding)
- Alpha Vantage API (real-time stock data)
- Slack API (trigger and response)

**Features:**
- Natural language queries
- Real-time market data
- Instant Slack responses
- Error handling for invalid tickers

## How to Use These Workflows

1. Download the JSON file from the `workflows/` folder
2. Open your n8n instance
3. Click "Import from File"
4. Select the JSON file
5. Configure your credentials (API keys, OAuth)
6. Activate and test
